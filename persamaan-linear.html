Materi Persamaan Linear

Persamaan linear adalah persamaan yang hanya melibatkan variabel berpangkat satu. Bentuk umum persamaan linear satu variabel adalah:

ax + b = 0

Di mana a dan b adalah konstanta, dan x adalah variabel.

1. Bentuk Umum

Persamaan linear satu variabel memiliki bentuk ax + b = 0.

2. Menyelesaikan Persamaan Linear

Tujuannya adalah mencari nilai x yang memenuhi persamaan. Caranya dengan memindahkan semua konstanta ke sisi kanan.

Contoh:

2x + 6 = 0

x = -3

3. Persamaan Linear Dua Variabel

Bentuknya: ax + by = c

Contoh:

2x + 3y = 12
